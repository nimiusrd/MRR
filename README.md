# MRR

## 仮想環境

### 作成

```bash
python3 -m venv .
```

### 有効化

```bash
. bin/activate
```

## install

```bash
pip3 install -r requirements.txt
```
## 実行

`-c`オプションは`src/config/`にあるpythonファイルを指定する．

```bash
python main.py -c two_same_2
```

## Jupyter

### 起動

```bash
jupyter notebook
```
基本的に`notebook/`でファイルを作成する．
